# IAM Agent UI

Single Nuxt app for the IAM Agent demo.

This UI will cover admin, employee, and customer support demo flows.

This app is set up with:

- Nuxt 4
- Tailwind CSS
- shadcn-vue
- pnpm

## Setup

Install dependencies:

```bash
pnpm install
```

Run the local dev server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```
